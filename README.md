## 주요 기능 및 구현 사항

[프로그래머스 과제 테스트](https://school.programmers.co.kr/skill_check_assignments/4) 내용을 바탕으로 구현하였습니다.

<br></br>
- **기능**
  - 검색어 입력을 통한 사진 검색
  - 랜덤 버튼 클릭시 무작위로 사진(1개) 확인 가능
  - 검색어 저장 및 검색어 히스토리 확인 가능(최대 5개)
 
- JavaScript 
  - 바닐라 자바스크립트로 컴포넌트 구현
  - 다크모드 지원
  - localStorage를 이용한 데이터 저장
 
- HTML
  - 시맨틱 마크업
 
- CSS
  - 미디어쿼리를 이용한 반응형 웹
  - fade in, fade out 효과
